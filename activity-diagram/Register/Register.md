:ขอบัตรประชาชนลูกค้า;

if (ลูกค้าอนุญาตให้ใช้บัตร?) then (ให้)
    :เสียบบัตรที่เครื่องอ่านบัตร;

    if (อ่านบัตรได้?) then (ไม่ได้)
        (A)
    else (ได้)
        :บันทึกข้อมูล;
    endif

else (ไม่ให้)
    (A)
    note right
        - ลูกค้า ไม่ต้องการให้บัตร/ไม่มีบัตร
        - เครื่องอ่านบัตรไม่สามารถอ่านได้
    endnote
    :กรอกข้อมูล;
    :บันทึกข้อมูล;
endif
